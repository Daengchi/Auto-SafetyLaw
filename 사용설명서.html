Chapter 1

시스템 소개

이 시스템은 안전보건 담당자가 수작업으로 처리하던 세 가지 업무를 자동화합니다.

📋
법규준수평가표 생성
국가법령정보센터에서 최신 법령 조문을 받아 평가표 Excel 파일을 자동 생성
🔍
제·개정 모니터링
법령의 조문 변경을 자동으로 감지하여 무엇이 바뀌었는지 파악
📧
이메일 알림
법령 변경이 감지되면 담당자 전원에게 자동으로 이메일 발송

자동화 흐름

매 평일 오전 9시 자동 실행
↓
국가법령정보센터 API 접속
→
법령 변경 여부 확인
↓ (변경 감지 시)
담당자 전원에게 이메일 자동 발송
법규준수평가표 생성(1번 기능)은 자동 실행되지 않습니다. 담당자가 필요할 때 직접 실행합니다.
Chapter 2

설치 방법

2-1. 시작 전 준비사항

항목내용확인
운영체제Windows 10 또는 Windows 11□
관리자 권한설치 시 관리자 계정으로 실행 필요□
인터넷 연결GitHub 및 law.go.kr 접속 가능 환경□
API 키국가법령정보센터에서 사전 신청 (3장 참고)□
Python이나 Git을 별도로 설치할 필요가 없습니다. 설치 프로그램이 자동으로 처리합니다.

2-2. 설치 단계별 안내

1
설치 파일 받기

담당자(PM)로부터 받은 설치.bat 파일을 바탕화면이나 임시 폴더에 저장합니다.

2
관리자 권한으로 실행

설치.bat 파일에서 마우스 오른쪽 버튼 클릭 → 관리자 권한으로 실행을 선택합니다.

일반 더블클릭으로 실행하면 오류가 발생할 수 있습니다. 반드시 관리자 권한으로 실행하세요.
3
설치 폴더 선택

폴더 선택 창이 열립니다. 프로그램을 설치할 위치를 선택하세요.

예) C:\ 를 선택하면 → C:\SafetyLaw 폴더에 설치됩니다.

선택한 폴더 안에 SafetyLaw 폴더가 자동으로 생성됩니다. 이미 SafetyLaw 폴더가 있는 경우 오류가 발생하므로 삭제 후 재설치하세요.
4
자동 설치 진행

검은 창(명령 프롬프트)이 열리며 자동으로 설치됩니다. 완료까지 수 분이 소요될 수 있습니다.

단계내용
[1/5]Portable Git 다운로드
[2/5]프로그램 코드 다운로드 (GitHub)
[3/5]Portable Git 설치 완료
[4/5]Python 및 필요 패키지 설치 (가장 오래 걸림)
[5/5]설정 파일 생성
설치 중 창을 닫거나 인터넷 연결이 끊기면 설치가 실패합니다. 완료될 때까지 기다려 주세요.
5
작업 스케줄러 등록

설치 완료 후 다음 질문이 나타납니다:

Register Task Scheduler? (Weekdays 09:00 auto-run) [Y/N]

Y를 입력하고 Enter를 누르면 매 평일 오전 9시에 법령 모니터링이 자동 실행됩니다.

N을 입력하면 자동 실행 없이 수동으로만 사용합니다.

자동 이메일 알림을 받으려면 Y를 선택하세요.
6
설치 완료 확인

"Installation complete!" 메시지가 표시되면 설치가 완료된 것입니다. Enter를 눌러 창을 닫습니다.

설치 폴더(예: C:\SafetyLaw)로 이동하면 프로그램 파일들이 생성된 것을 확인할 수 있습니다.

Chapter 3

초기 설정 (설치 후 1회만)

3-1. API 키 발급

국가법령정보센터 Open API 키를 무료로 발급받아야 합니다.

1
법령정보 Open API 페이지 접속

웹 브라우저에서 아래 주소로 이동합니다:

https://open.law.go.kr/LSO/main.do
2
회원 가입 및 로그인

국가법령정보센터 회원 가입 후 로그인합니다.

3
API 신청

"Open API 신청" 버튼을 클릭하여 사용 목적을 입력하고 신청합니다.

서버 IP 주소 입력란에 이 PC의 공인(외부) IP 주소를 입력해야 합니다. (확인 방법은 다음 단계 참고)

4
이 PC의 공인(외부) IP 주소 확인 방법

웹 브라우저(크롬·엣지)를 열고 네이버에서 내 아이피 주소 확인을 검색합니다.

화면 맨 위에 표시되는 IP 주소(예: 211.xx.xx.xx)를 API 신청 시 입력합니다.

ipconfig로 나오는 내부 IP(192.168.x.x, 10.x.x.x 등)는 등록하면 안 됩니다. 외부에서 보이지 않아 검증에 실패합니다. 반드시 위 방법으로 확인한 공인 IP를 등록하세요.
PC의 공인 IP가 변경되면 법령정보센터에 다시 접속하여 IP를 재등록해야 합니다. 회사 네트워크가 여러 공인 IP로 나가거나 IP가 수시로 바뀌면 사내 IT팀에 고정 공인 IP를 문의하세요.
5
인증키 확인

신청 승인 후 발급된 인증키(영문·숫자 조합)를 복사해 둡니다. 다음 단계에서 사용합니다.

3-2. API 키 입력 (.env 파일)

발급받은 API 키를 설정 파일에 입력합니다. 두 곳에 모두 입력해야 합니다.

① 1번 프로젝트 API 키 입력

1
파일 위치로 이동

탐색기에서 설치 폴더를 열고 01. 법규 데이터 관리 폴더로 이동합니다.

.env 파일이 보이지 않으면 탐색기 상단 메뉴 → 보기 → 숨긴 항목에 체크하세요.
2
파일 열기

.env 파일에서 마우스 오른쪽 클릭 → 연결 프로그램 → 메모장

3
API 키 입력

아래와 같이 LAW_API_KEY= 뒤에 발급받은 인증키를 붙여넣습니다.

LAW_API_KEY=여기에인증키붙여넣기

저장 (Ctrl+S) 후 닫습니다.

② 2번 프로젝트 API 키 입력

위와 동일하게 02. 제개정사항 모니터링 폴더의 .env 파일에도 동일한 API 키를 입력합니다.

3-3. 법령 목록 설정 (laws.json)

모니터링하고 싶은 법령 목록을 설정합니다. 기본으로 아래 15개 법령이 포함되어 있습니다.

산업안전보건법
산업안전보건기준에 관한 규칙
중대재해 처벌 등에 관한 법률
위험물 안전관리법
고압가스 안전관리법
소방시설 설치 및 관리에 관한 법률
화재의 예방 및 안전관리에 관한 법률
화학물질관리법
잔류성오염물질 관리법
환경오염시설의 통합관리에 관한 법률
화학물질의 등록 및 평가 등에 관한 법률
폐기물관리법
물환경보전법
대기환경보전법
토양환경보전법

법령 삭제 방법

1
파일 열기

설치 폴더 최상위의 laws.json 파일을 메모장으로 엽니다.

2
불필요한 법령 삭제

삭제할 법령이 적힌 줄 전체를 지웁니다. 마지막 법령에는 쉼표(,)가 없어야 합니다.

[
  "산업안전보건법",
  "중대재해 처벌 등에 관한 법률"    ← 마지막 항목엔 쉼표 없음
]
JSON 형식에서 마지막 항목 뒤에 쉼표가 있으면 오류가 발생합니다. 주의하세요.
3
저장

Ctrl+S로 저장합니다.

3-4. 이메일 알림 설정 (선택사항)

법령 변경 시 이메일 알림을 받으려면 아래 두 파일을 설정합니다. IT 담당자의 도움이 필요할 수 있습니다.

① SMTP 서버 설정 (.env)

03. 법규 제개정 이메일 알림 폴더의 .env 파일을 메모장으로 열어 수정합니다.

항목설명예시
SMTP_HOST사내 이메일 서버 주소mail.company.com
SMTP_PORT서버 포트 (IT 담당자에게 문의)25
SMTP_USER발신 계정 (없으면 빈칸)noreply@company.com
SMTP_PASSWORD비밀번호 (없으면 빈칸)
SMTP_FROM발신자로 표시될 이메일safety@company.com
SMTP_USE_TLS암호화 사용 여부false

② 수신자 목록 설정 (recipients.json)

같은 폴더의 recipients.json 파일을 메모장으로 열어 이메일 주소를 입력합니다.

[
  "담당자1@회사.com",
  "담당자2@회사.com",
  "팀장@회사.com"
]
이메일 주소를 추가할 때마다 위와 같은 형식으로 한 줄씩 추가합니다. 마지막 항목에는 쉼표가 없어야 합니다.
Chapter 4

사용 방법

4-1. 법규준수평가표 처음 만들기

파일: 01_신규생성.bat (설치 폴더 최상위에 위치)

언제 사용하나요?

  • 처음 설치 후 평가표를 처음 만들 때
  • laws.json에 새로운 법령을 추가했을 때
  • 전체 법령의 평가표를 새로 만들고 싶을 때
1
실행

설치 폴더에서 01_신규생성.bat 파일을 더블클릭합니다.

2
진행 과정 확인

검은 창이 열리며 법령별 처리 현황이 표시됩니다. laws.json의 법령 수에 따라 수십 분이 걸릴 수 있습니다.

API 사용량 제한으로 인해 법령당 약 10~30초 소요됩니다. 창을 닫지 말고 기다려 주세요.
3
결과 확인

완료 후 "아무 키나 누르세요" 메시지가 나타나면 Enter를 누릅니다.

결과 파일 위치: [설치경로]\01. 법규 데이터 관리\output\법규준수평가표_YYYYMMDD.xlsm

4-2. 기존 평가표 업데이트

파일: 01_업데이트.bat (설치 폴더 최상위에 위치)

언제 사용하나요?

  • 이미 만들어 둔 평가표 파일을 최신 법령 내용으로 갱신할 때
  • 특정 법령만 업데이트하고 싶을 때
1
실행

01_업데이트.bat 파일을 더블클릭합니다.

2
파일 선택

파일 선택 창이 열립니다. 업데이트할 법규준수평가표 파일(.xlsm)을 선택하고 열기를 클릭합니다.

3
완료 확인

업데이트가 완료되면 같은 위치에 파일이 갱신됩니다.

4-3. 제개정사항 모니터링 (리포트 생성)

파일: 02_제개정모니터링.bat (설치 폴더 최상위에 위치)

언제 사용하나요?

  • laws.json 법령의 제·개정 변경 여부를 직접 확인하고 싶을 때
  • 변경 내역을 Excel 리포트로 받아 보고 싶을 때
1
실행

설치 폴더에서 02_제개정모니터링.bat 파일을 더블클릭합니다.

2
진행 과정 확인

검은 창에 법령별 상태(신규 등록 / 개정됨 / 변경 없음)가 표시됩니다.

3
결과 확인

결과 파일 위치: [설치경로]\02. 제개정사항 모니터링\output\법규_제개정_모니터링_YYYYMMDD_HHMM.xlsx

이 기능은 이메일을 발송하지 않습니다. 변경 시 자동으로 이메일을 받으려면 5장의 자동 모니터링을 이용하세요.
02. 제개정사항 모니터링 폴더의 .env에 API 키가 입력되어 있어야 하고, 법령정보센터에 이 PC의 공인 IP가 등록되어 있어야 조회됩니다. (3장 참고)
Chapter 5

자동 모니터링 설정

설치 시 작업 스케줄러 등록(Y)을 선택했다면 이미 자동화가 완료된 상태입니다. 매 평일 오전 9시에 아래 작업이 자동 실행됩니다.

① 최신 코드 자동 업데이트
→
② 법령 변경 확인
→
③ 변경 시 이메일 발송
↓
실행 기록 → logs\실행로그.log 저장

작업 스케줄러 수동 등록 방법

설치 시 N을 선택했거나 자동 등록에 실패한 경우:

1
작업 스케줄러 열기

Windows 검색창에 "작업 스케줄러" 검색 후 실행합니다.

2
새 작업 만들기

오른쪽 패널에서 "기본 작업 만들기..."를 클릭합니다.

3
트리거 설정

반복 실행 선택 → 매주 → 월·화·수·목·금 체크 → 시간: 오전 9:00

4
동작 설정

프로그램 시작 선택 → 찾아보기 → [설치경로]\법규모니터링_자동실행.bat 선택

실행 로그 확인

자동 실행 기록은 [설치경로]\logs\실행로그.log 파일에 저장됩니다. 메모장으로 열어 확인할 수 있습니다.

이메일 알림이 오지 않는다면 로그 파일에서 오류 메시지를 확인하세요.
Chapter 6

자주 묻는 질문 (FAQ)

실행해도 아무 파일도 생성되지 않아요.
API 키가 올바르게 입력되었는지 확인하세요. .env 파일을 메모장으로 열어 LAW_API_KEY= 다음에 인증키가 있는지 확인합니다. 또한 law.go.kr에서 이 PC의 공인(외부) IP 주소가 등록되어 있는지 확인하세요.
"사용자 정보 검증에 실패하였습니다" 오류가 떠요.
이 PC의 공인(외부) IP 주소가 법령정보센터에 등록되지 않은 경우입니다. 브라우저에서 네이버에 내 아이피 주소 확인을 검색해 IP를 확인한 뒤, 법령정보센터 API 신청 정보에 그 IP를 등록하세요. (3-1장 참고) ipconfig의 내부 IP가 아니라 공인 IP여야 합니다.
프로그램 창이 켜졌다가 바로 꺼져요.
.env 파일에 API 키가 입력되지 않았을 가능성이 높습니다. 3-2장을 참고하여 API 키를 입력하세요.
이메일이 오지 않아요.
① 법령 변경이 실제로 없었을 수 있습니다. ② 03. 법규 제개정 이메일 알림\.env의 SMTP 설정을 확인하세요. ③ recipients.json에 수신자 이메일이 올바르게 입력되었는지 확인하세요. ④ IT 담당자에게 SMTP 서버 정보를 문의하세요.
법령을 추가하고 싶어요.
laws.json 파일을 메모장으로 열어 법령명을 추가합니다. 형식: "법령명", (마지막 항목은 쉼표 없음). 추가 후 01_신규생성.bat을 실행하면 해당 법령의 평가표가 생성됩니다.
설치 중 오류가 발생했어요.
① 관리자 권한으로 실행했는지 확인하세요. ② 인터넷 연결을 확인하세요. ③ SafetyLaw 폴더가 이미 존재한다면 삭제 후 재설치하세요.
프로그램이 업데이트됐다고 하는데 어떻게 받나요?
별도 작업 불필요합니다. 01_신규생성.bat, 01_업데이트.bat, 자동 실행 배치 파일 중 어느 것을 실행해도 최신 코드를 자동으로 받아옵니다.
다른 PC에도 설치하고 싶어요.
담당자(PM)에게 설치.bat 파일을 요청하세요. 해당 PC에서 API 키를 별도로 발급받거나, 기존 API 키에 새 PC의 공인(외부) IP 주소를 추가 등록해야 합니다.
